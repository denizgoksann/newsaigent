# NewsAIgent

NewsAIgent habercilerin işlerini kolaylaştırmak için kullanıcı dostu, başarı oranı yüksek ve haberciyi yormadan haber oluşturabilecekleri, ellerindeki habere spot ve/veya başlık oluşturabilecekleri, bazı büyük haber sitelerindeki akışı kolaylıkla takip edebilecekleri ve haberleri yayınlayarak kendi ekosistemlerindeki habercilerle paylaşabilecekleri yapay zeka destekli bir tool.

## Gereksinimler
- laravel v10
- php v8.2>
- composer
- MySQL
## Kurulum
- Repoyu indirin.
- Xampp klosörünüzdeki htdocs içerisinde yükleyin.
- MySQL içerisinde ai adında şema oluşturun.
- .env dosyası içerisinde bulunan DB_USERNAME ve DB_PASSWORD alanlarını kendinize göre güncelleyin.
- controllerların içerisindeki API_KEY alanlarını kendi api keyleriniz ile güncelleyin.
- MySQL inizi ve Xampp içerisinden apache modülünü çalıştırarak sistemi kullanabilirsiniz.
